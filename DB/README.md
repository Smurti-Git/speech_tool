# Speech Recognition Project

This project is a simple web application that uses the Web Speech API for speech recognition. It allows users to start and stop speech recognition, display the recognized text, and copy the text to the clipboard.

## Features

- **Speech Recognition**: Start and stop speech recognition using a button.
- **Text Display**: Display the recognized text in a text area.
- **Copy Text**: Copy the recognized text to the clipboard.
- **Clear Text**: Clear the text area with a button click.

## Getting Started

To get started with this project, follow these instructions.

### Prerequisites

Make sure you have the following:

- A web browser that supports the Web Speech API (e.g., Google Chrome).
- Microphone access enabled for the browser.
